Predição de Elegibilidade de Empréstimos

Visão Geral: Este projeto usa modelos de Árvore de Decisão e Floresta Aleatória para prever se um cliente provavelmente terá um empréstimo aprovado. A ideia é identificar os fatores que influenciam essa decisão e comparar diferentes modelos.

Objetivo: Criar um fluxo completo de Machine Learning: preparar dados, explorar informações, treinar modelos, avaliar resultados e interpretar o que realmente importa para a tomada de decisão.
